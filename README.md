# ダウンロードの際
ダウンロードする場合は<br>

PC　右のReleaseから<br>

スマホ　下or上のReleaseから<br>

下の〇〇.mcaddon(.mcpack)っていう青い奴をクリックすると、<br>
ダウンロードできます！
